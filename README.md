# DuckcraftUI

A modern, Dragonflight-inspired user interface overhaul built specifically for the **Duckcraft** server running on the World of Warcraft Wrath of the Lich King client (3.3.5a).

## Features
* **Modern Aesthetics:** Brings the clean, updated look of the retail Dragonflight UI to the legacy WotLK client.
* **Customizable Fading & Visibility:** Built-in configuration options to manage interface fading and element visibility.
* **Auto-Loot Configuration:** Tailored auto-loot settings optimized for the Duckcraft server environment.

## Installation

1. Click the green **Code** button at the top of this repository and select **Download ZIP**.
2. Extract the downloaded ZIP file.
3. Open the extracted folder and locate the two addon directories:
   * `DuckcraftUI`
   * `DuckcraftUI_Options`
4. Copy both folders into your WoW AddOns directory:
   `World of Warcraft\_classic_\Interface\AddOns\` *(or your equivalent 3.3.5a path)*
5. Launch World of Warcraft, ensure the addons are enabled in the character select screen, and log in!

## Configuration
In-game options for fading, visibility, and other UI tweaks can be found in the standard Interface Options menu under the AddOns tab, provided by the `DuckcraftUI_Options` module.
